# 신한 SOL 목업 웹앱

신한은행 SOL 앱 화면(금융 메인 / 계좌상세 / 계좌관리)을 재현한 정적 웹앱입니다.
순수 HTML/CSS/JS 단일 파일(`index.html`)이라 별도 빌드 없이 Vercel에 바로 배포됩니다.

## 기능
- 금융(은행) 메인 → 계좌상세 → 계좌관리 화면 전환
- 모든 버튼 클릭 시 목업 페이지로 이동
- 매월 25일 기준 자동 적립(+150,000원) 계산
- 금액 숨김 토글, 계좌번호 복사 등

## 로컬 실행
`index.html` 파일을 브라우저로 열기만 하면 됩니다.

## 배포 (Vercel)
GitHub 저장소에 올린 뒤 Vercel에서 Import → Deploy 하면 끝입니다.
프레임워크 설정은 **Other / 빌드 명령 없음**으로 두면 됩니다.
